A responsive React application that allows users to browse products, view details, and create new product entries.

Demo:  https://bostafrontend.netlify.app/


-Data is fetched from FakeStoreAPI

Pages:
-Products: displays the fetched products
-Product Details: displays the selected product
-CreateProduct: allows the user to create a new product using the POST API

Features:
-sorting: sorting the products by price (ascendingly & descendingly) & by Category
-Pagination: Each page holds 10 products only
-Responsive UI using Bootstrap
-Data Validation and type checks


Tech Stack:
-React.js
-React Router Dom
-Axios
-Bootstrap
